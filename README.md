<h1>統一碼漢字表</h1>

<p>統一碼裏該有卻沒有的字：</p>
<ul>
<li>人部：
<ul>
<li>從人息聲（左人右息）</li>
</ul>
</li>

<li>口部：
<ul>
<li>兩個不同聲符（真、眞）的嗔字</li>
</ul>
</li>


<li>土部：
<ul>
<li>墟的本字，左土右虛（不是虚；這也許是字體的問題，需要兩個不同的字）</li>
</ul>
</li>

<li>寸部：
<ul>
<li>應該把寽（爪寸）、將的右半（肉寸）分開</li>
</ul>
</li>


<li>山部：
<ul>
<li>從山嗇聲（左山右嗇）</li>
</ul>
</li>

<li>心部：
<ul>
<li>愉的本字，從心，兪（不是俞）聲</li>
<li>愈的本字，上兪下心（這也許是字體的問題，需要兩個不同的字）</li>
</ul>
</li>

<li>手部：
<ul>
<li>拼的本字，從手幷聲</li>
</ul>
</li>

<li>犬部：
<ul>
<li>兩個不同聲符（俞、兪）的㺄字</li>
</ul>
</li>


<li>目部：
<ul>
<li>直的本字，从﹂从十从目（這也許是字體的問題，需要兩個不同的字）</li>
</ul>
</li>
<li>舟部：
<ul>
<li>䒂去掉中間的爿</li>
</ul>
</li>

<li>豸部：
<ul>
<li>兩個不同聲符（俞、兪）的貐字</li>
</ul>
</li>


<li>足部：
<ul>
<li>兩個不同聲符（开、幵）的趼字</li>
</ul>
</li>
